### Vue
    vue-cli  4版本的脚手架可以选择vue3版本的安装



## Vite(快的意思)是一个http服务器(服务Vue3的开发框架) ：
    1.可以在单文件中书写ES6 的语法
    2.支持热更新(请求的内容才会被打包/更新)
    3.Rollup打包

# 通过vite脚手架创建项目
    npm install -g create-vite-app
    create-vite-app <project-name>
    cd <project-name>
    npm install
    npm run dev

## 通过vue-cli和vite创建的项目目录结构基本一致
